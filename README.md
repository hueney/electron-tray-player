## MP3/Radio player - system tray application (Electron + Vue.js + Framework7)

### Installation
`npm install`

`npm start`

### Packaging
`npm run build`

### Known issues
* Solution for [bug] gyp ERR! configure error ([#3][i3]): run `npm rebuild` or `yarn rebuild`
